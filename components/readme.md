# Components

Primitive UI components with theme integration for expo/react-native applications.

## components reference

| component   | description                                      |
|-------------|--------------------------------------------------|
| [Avatar](/components/avatar/readme.md)         | user image display with fallback initials        |
| [Badge](/components/badge/readme.md)           | status indicator with color schemes             |
| [Box](/components/box/readme.md)               | generic view wrapper with shorthand props       |
| [Button](/components/button/readme.md)         | interactive button with variants               |
| [Card](/components/card/readme.md)             | surface container with variants                |
| [Divider](/components/divider/readme.md)      | separator line                                 |
| [Flex](/components/flex/readme.md)            | flexbox layout primitive                       |
| [Icon](/components/icon/readme.md)             | icon wrapper using octicons                    |
| [Input](/components/input/readme.md)           | text input with variants                      |
| [Pressable](/components/pressable/readme.md)  | animated pressable wrapper                     |
| [Spinner](/components/spinner/readme.md)       | loading indicator                             |
| [Text](/components/text/readme.md)             | typography primitive                          |
